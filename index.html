How to use

  • 1.Press the number button to select the first value
  • 2.Press math operation
  • 3.Press the number button to select the second value
  • 4.Press the equal sign (to get the result you can not press any other operator)
  • 5.Get result

Notice: If you make a mistake with a finite number and want to delete it, click the button CE. This will allow you to clear the last number, provided that your number does not exceed the number of characters greater than 3, otherwise the number will be reset.
